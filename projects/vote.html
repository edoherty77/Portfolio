Vote-ify

Previous Next

Vote-ify brings Democracy to playlists. Using the Spotify API, users can listen to and contribute songs to 12 playlists, all curated and managed by the community. Carefully creating a Spotify API endpoint, I implemented a search bar, where a user can search and 'suggest' a song. It is then sent into a queue where it has two weeks to earn enough votes to be officially added to the playlist.
